# Pengembangan-Model-Segmentasi-Pelanggan-Berbasis-Machine-Learning-untuk-Optimalisasi-Promosi
Proyek ini bertujuan untuk menganalisis perilaku pelanggan menggunakan pendekatan RFM (Recency, Frequency, Monetary) dan Machine Learning (Clustering). Hasil analisis digunakan untuk mengelompokkan pelanggan ke dalam segmen yang bermakna guna mendukung pengambilan keputusan bisnis.
